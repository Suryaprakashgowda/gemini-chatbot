# AI-Powered Gemini Chatbot using Streamlit

A modern conversational AI chatbot built using **Streamlit** and **Google Gemini 2.5 Flash**. This application enables users to interact with Google's advanced Generative AI model through an intuitive web-based chat interface.

---

## 🚀 About The Project

The AI-Powered Gemini Chatbot is a lightweight and user-friendly web application that allows users to communicate with Google's Gemini AI model in real time. The chatbot maintains conversation history during a session and provides intelligent responses to user queries.

### ✨ Key Features

* 🤖 Powered by Google Gemini 2.5 Flash
* 💬 Real-time conversational interface
* 📜 Session-based chat history
* 🎨 Clean and responsive Streamlit UI
* 🔒 Secure API key management using environment variables
* ⚡ Fast response generation

## 📋 Prerequisites

Before running this project, ensure you have:

* Python 3.9 or above
* Google Gemini API Key
* Streamlit

---

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/suryaprakashgowda/Gemini-ChatBot-main.git
cd Gemini-ChatBot-main
```

### 2. Install Dependencies

```bash
pip install streamlit google-genai python-dotenv
```

### 3. Configure API Key

Create a `.env` file in the project root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run the Application

```bash
streamlit run app.py
```

The application will start at:

```text
http://localhost:8501
```

## 📖 Usage

1. Open the application in your browser.
2. Enter your message in the chat input box.
3. Press Enter to send your message.
4. Receive AI-generated responses instantly.
5. Continue the conversation naturally.
